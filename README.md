## Installation

To play [Rock Paper Scissor], follow these simple steps:

1. Clone the repository to your local machine.
    ```bash
    git clone https://github.com/your-username/game-repository.git
    ```

2. Navigate to the game directory.
    ```bash
    cd game-repository
    ```

3. Run the game.
    ```bash
    python game.py
    ```
